## Requirements

- [Node.js](http://nodejs.org/)
- [Grunt.js](http://gruntjs.com/)
- [Bower](http://bower.io/)
- [Jekyll](http://jekyllrb.com/)

## Getting things running locally

    $ git clone git@github.com:jonsuh/jonsuh.com.git
    $ cd jonsuh.com
    $ npm install
    $ bower install
    $ grunt build
